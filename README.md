# driving-exam-app

**Spring Boot  JWT Authentication**

This project is a driving Licensing exam backend Springboot project.It implemented RESTful APIs,Spring Security using JWT authentication


**Developer Notes**

The project makes use of POSTGRES, pgAdmin4 and  Docker images for database support.

git clone https://github.com/kyeiwrk23/driving-exam-app.git

cd driving-exam-app

Open in your favorite editor supporting Java (I used Intellij)
Note that if you decide to use this project, I generated  JWT secret key in the project by using KeyGenerator class from javax.crypto package. 

Also, there are controllers for user,exam and questions to help you verify role authorization.
